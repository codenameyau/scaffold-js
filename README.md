# scaffold-js

Node CLI for creating the following files:

- .gitignore
- .npmignore
- .eslintrc
- .jshintrc
- .editorconfig

### Installation

```
sudo npm install -g scaffold-js
```

To use the ES6 babel-compiler for eslint, you'll need to install:

```
sudo npm install -g babel babel-eslint eslint
```

### Usage

```
scaffold-js
```
